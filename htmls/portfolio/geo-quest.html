Project

GeoQuest

Developed fundamental services of a location-based game.

Includes: User management (Accounts register and management (OAuth supported: Login/out via google/FB accounts), Game files storage management (Upload/download/delete games on server running MongoDB) and Game format validator..etc.

Server-side is developed in Node.js, Express. Android client is using PhoneGap.

The project development process was running in Scrum-ban and Agile practices including TDD/BDD (Mocha), Continuous Integration (TravisCI), Pair Programming..etc.

Source: Server Client
TAG: Node.js, Express, MongoDB, Mocha (TDD/BDD), TravisCI (CI), PhoneGap, EJS (template engine), OAuth, Validator
