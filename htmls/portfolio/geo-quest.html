Project

GeoQuest

Developed fundamental services of a location-based game. Server-side is developed in Node.js, Express. Android client and web is using PhoneGap and jQuery Mobile.

Including: User management (Accounts register and management (OAuth supported: Login/out via google/FB accounts), Game files storage management (Upload/download/delete games on server running MongoDB) and Game format validator..etc.

The project development process was running in Scrum-ban and Agile practices including TDD/BDD (Mocha), Continuous Integration (TravisCI), Pair Programming..etc.

Source: Server Client
Keywords: Agile, Node.js, Express, MongoDB, Mocha (TDD/BDD), TravisCI (CI), PhoneGap, jQuery Mobile, EJS (template engine), OAuth, Validator
