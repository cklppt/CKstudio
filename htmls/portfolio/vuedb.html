Project

Vuedb

電影資料庫

https://vuedb.com/

自家產品 Vuedb 電影資料庫。蒐集了 imdb 一百五十多萬筆電影，以及關聯的數百萬上千萬筆演員、導演、編劇等 staff 資料，再交叉比對時光網 mtime、豆瓣，以及 wikipedia 補足缺少中文資料，並蒐集各種電影 Facebook 粉絲團熱門貼文。希望成為華文區資料最齊全的電影資料庫。負責後端主程式/架構/資料庫設計。

  • 開了十台 Vultr 機器 (為了 IP)，寫 node.js crawler 取得 imdb/時光網/豆瓣/wikipedia 數百萬筆中英文電影與 staff 資料
  • 建立簡單 SQL HA clusters 以及 DB 速度優化，提升 FTS 全文搜尋響應速度
  • 寫自動維護工具程式與 API 給站台維運人員使用，例如一鍵比對同名中英文電影，一鍵取得各大網站電影資料，一鍵取得各站評分等
  • 設計實作彈性化維護工具模塊的 node.js 服務架構，包含本地異地 (異地IP Pool)，底層 DB engine，Filter，同步異步，選定的更新維護 task 模塊，皆可彈性搭配執行，批次大量維護。功能像 IP pool 自動切換使用的異地 IP，模組化的各種 task (抓評分/找中文電影名稱/抓介紹/抓演員導演等) 與異步 task runner，可切換底層 DB 的 ORM，後台 web socket 即時顯示進度等
Keywords: Node.js, PHP/Laravel, Architecture, Database(MySQL, MongoDB, Redis), API, Modules, crawlers, maintenance tools
