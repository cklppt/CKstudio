Project

LandingShark

https://www.leadxo.com/

內部 Drag-n-Drop 快速建站產品。全端開發。Html builder 是買來的半成品套件，但全站架構與流程皆自行設計開發。軟體需求是，以低資源管理 host 大量網站，一鍵發佈，快速上傳更新，立即上線看見成果。

  • 利用 AWS S3 的 static website hosting，使 host 大量網站也完全不占伺服器資源
  • builder提供給使用者的預設 design blocks 所需 assests (js, css, jpgs, fonts) 都已 host 在 S3 上設 CORS 所有網站共用，不須重複上傳
  • 僅修改更新網站時需要使用伺服器資源，而也只需上傳html與使用者自訂內容，資料量小，上傳速度極快

使用者按下發佈：使用 builder 產生 html 後，與自訂資源打包成 zip 上傳至 AWS S3，自動設定 CORS 與 static hosting，並即時修改 Cloudflare CNAME，便可立即在 user.landingshark.com 看見成果。

Keywords: PHP/Laravel, Javascript, AWS S3 API (static website hosting, CORS), Cloudflare API
