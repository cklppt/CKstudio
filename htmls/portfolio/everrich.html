Project

Everrich

https://www.everrich.com/

台灣最大免稅電商網站，結合線上線下銷售。如果你在台灣機場購買過免稅商品，那你就使用過我們的服務。 使用 Prestashop + Laravel 自行改造之架構，負責中後期開發與追加需求，後續維護

  • 開發複雜優惠邏輯 (買幾件/同時買/同品牌/A+B/滿額/贈品等)，改善演算法增進效能
  • 開發前端 ShopWithFriend，使用戶能夠發邀請給朋友使用同一購物車共享優惠，使用 web socket 即時進行多人線上同購物車購物
  • 與線下銷售點同步，產生 pdf 訂單與後台取貨確認、POS列印等
  • 部屬在 AWS 上，度過 iPhone 搶購時的壓力測試
Keywords: E-commerce, PHP/Laravel, Prestashop, Javascript, web scoket, AWS
