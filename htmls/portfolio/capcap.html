Project

Capcap

網站截圖服務

http://capcap.tenten.co/captureSite?url=www.gitlab.com&no_queue

Tenten 內部產品，網站截圖服務。三層多行程結構。DB/Queue 層接到截圖請求先找 DB 是否 hit，若截圖夠新則回傳，否則加進 queue。node.js 使用 cluster，Master 層，負責管理 workers 並從 queue 拉工作來分派，監控 worker 是否 busy，若 worker 當掉則發訊息 kill worker 重開。Workers 層使用 xvfb (headless x server) + nightmare 真實 browser 實現截圖，截圖完成再回報 master。

Keywords: Node.js, PHP/Laravel, cluster
