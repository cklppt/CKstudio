Project

Kancolle Shooter

A fan-made non-commercial 3rd-Person Shooter game of of Kancolle (Kantai Collection, 艦隊これくしょん, a Japanese web game by KADOKAWA.)

It is self-developed as a practice of fast prototype, or MVP (Minimum Viable Product), purely out of interest and only privately shared between friends and enthusiastic fans for testing and development purpose.

The game is written in C# and developed using Unity3D.
