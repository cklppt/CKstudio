Project

Bio-safty lab Simulator

A infectious biological laboratory simulator for accident handling training.

Proposed and designed the Kinect interface for controlling the avatar via motion capture. Implemented a prototype which supports using body movement to move around, hand/gesture detection to grasp things.

Developed in C#, C++, OpenCV and Unity3D.
