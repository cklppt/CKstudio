Front-end Animations

一些隨手寫的前端動畫

偶爾會需要寫一些前端動畫，少數有放到 codepen 上。

  • LaunchKitty (codepen)

    • 使用 anime.js。雖然是 toy animation，還是寫了模組化的動畫模塊與 task runner，只要使用封裝好的 API 填上動畫參數，就能輕鬆地將 kitty 任意發射。

  • Waves (codepen)

    • 簡單的複數波形動畫，使用 p5.js (Processing 的 js 框架)。當初有考慮發佈到 github，但還有一些功能想要追加，也沒有特別考慮過效能，後來就閒置了。

Keywords: Front-end, anime.js, p5.js (Processing)
